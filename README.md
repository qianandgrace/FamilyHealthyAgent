# 数据收集与清理
## 疾病库
下载卫建委指南，并使用mineru解析清理文档，转化为md文件
因为做的是中文agent,所以去除英文
丁香医生常见疾病讲解
爬虫提取 30个科室常见的197个疾病

## 红旗数据库
搜集最常见的50种紧急症状

## 分诊数据库
搜集常见的症状，根据严重等级 年龄等确定是否需要去医院

# 方案选型

使用agentic 的rag
